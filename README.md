# Titanic Data Science Project

## Overview

This project was completed as part of the **BCS 404: Introduction to Data Science with Python** course at **Accra Technical University**.

The project applies Python to perform:

- Data Acquisition
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Machine Learning using Logistic Regression

The objective is to analyze the Titanic dataset and develop a predictive model for passenger survival.

---

## Dataset

Source:

https://www.kaggle.com/competitions/titanic/data

The dataset contains demographic and passenger information from the RMS Titanic, including:

- Passenger Class
- Sex
- Age
- Fare
- Family Information
- Survival Status

## Machine Learning Model

A Logistic Regression classifier was developed to predict passenger survival.

The model achieved an accuracy of approximately **77%** on the testing dataset.
